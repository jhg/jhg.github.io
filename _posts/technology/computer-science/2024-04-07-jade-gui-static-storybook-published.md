**[Jade GUI][jade-gui]**, a simple [design system][design-system] with an [Storybook][storybook]
  for my own use.

## Published an static Storybook of Jade GUI

Automatically when [Jade GUI][jade-gui] has new commits a static
  [GitHub Page][github-pages] will be published, with the
  [Storybook of Jade GUI][jade-gui-storybook]. With that [Storybook][storybook]
  is easy to see current state of my components library.

It uses [Web Components][web-components] built with [Lit framework][lit-dev].
