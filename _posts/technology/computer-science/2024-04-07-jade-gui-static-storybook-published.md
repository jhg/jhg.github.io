**[Jade UI][jade-ui]**, a simple [design system][design-system] with an [Storybook][storybook]
  for my own use.

## Published an static Storybook of Jade UI

Automatically when [Jade UI][jade-ui] has new commits a static
  [GitHub Page][github-pages] will be published, with the
  [Storybook of Jade UI][jade-ui-storybook]. With that [Storybook][storybook]
  is easy to see current state of my components library.

It uses [Web Components][web-components] built with [Lit framework][lit-dev].
